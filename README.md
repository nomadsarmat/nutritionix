# Nutritionix Macro Tracker #

Just a simple way to track macros...if this app ever gets finished.
Uses:

*  Firebase
*  Angular
*  Grunt
*  Coffee-Script

## Installation ##

*  `npm install`
*  `grunt && grunt watch`
*  `./scripts/web-server.js`
*  Figure out how to make your own config.js, you can't have mine
